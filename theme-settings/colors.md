# Colors settings

Define your store's color palette — from layout and typography to buttons, overlays, and pickup availability indicators.

`[SCREENSHOT: Full Colors panel in Theme settings]`

---

## Layout

Global colors for the page background, text, and UI elements.

| Setting | Description |
|---------|-------------|
| **Background** | Default background color for all pages. |
| **Text** | Default text color throughout the store. |
| **Compare-at price color** | Choose **Theme default** to use the standard text color, or **Custom** to set a specific color for strikethrough prices. |
| **Custom compare-at price color** | Visible only when Compare-at price color is set to Custom. Set via the color picker. (v13) |
| **Popup background** | Background color for dropdowns, search panel, and modal windows. Choose a color close to your layout background for a seamless look. |
| **Dim secondary text** | Reduces the brightness of secondary text colors throughout the theme. |
| **Enable accent color for headings** | Changes all heading colors to match the accent color set in Buttons and links. |

---

## Buttons and links

Controls the color system for all interactive elements — buttons, links, and accents.

| Setting | Description |
|---------|-------------|
| **Base** | Text color for **Solid** style buttons. |
| **Accent** | Background color for **Solid** buttons; text and outline color for **Outlined** buttons; text color for **Link** style buttons. This color also drives highlights and interactive elements across the theme. |
| **Hover** | Background color on hover for **Solid** buttons; text and outline color on hover for **Outlined** buttons; text color on hover for **Link** buttons. |

---

## Drawers and popups overlay

Controls the overlay that appears behind drawers (cart, menu, store selector) and modal windows.

| Setting | Description |
|---------|-------------|
| **Color** | Color of the overlay backdrop. |
| **Opacity** | Opacity of the overlay (0–100%). Lower values create a more transparent backdrop. |
| **Background blur** | Blur intensity applied to the content behind the overlay (in px). Set to 0 to disable. |

---

## Burger button

| Setting | Description |
|---------|-------------|
| **Fill** | Adds a background fill behind the burger menu icon. |
| **Shadow** | Intensity of the drop shadow on the burger icon (0–100%). |

---

## Close button

| Setting | Description |
|---------|-------------|
| **Shadow** | Intensity of the drop shadow on close/dismiss buttons (0–100%). |

---

## Extra

| Setting | Description |
|---------|-------------|
| **Rating** | Color for star ratings and review scores. |
| **Error** | Color for error messages and validation alerts. |
| **Success** | Color for success messages and confirmations. |
| **Use accent color for text selection** | Replaces the browser's default text selection highlight with your accent color. |

---

## Pickup availability

Colors for the store pickup availability indicators shown on product cards and product pages. Requires [Store Selector](../features/store-selector.md) to be configured.

| Setting | Description |
|---------|-------------|
| **Available text** | Text color for "Available at [store]" labels. |
| **Available icon** | Color of the availability indicator dot when the product is available. |
| **Out of stock text** | Text color when the product is unavailable at the selected location. |
| **Out of stock icon** | Color of the indicator dot when the product is unavailable. |
