# How do I create a FAQ page with expandable questions?

1. Go to **Shopify Admin → Online Store → Pages**
2. Click **Add page** and give it a title (e.g. "FAQ")
3. On the right side, assign the **FAQ template** to the page

{% hint style="info" %}
If you don't see the FAQ template in the dropdown, you need to create it first.

See: [A page template isn't showing →](a-page-template-isnt-showing-in-the-shopify-admin-dropdown.md)
{% endhint %}

1. Open **Theme Editor → Pages → FAQ template**.
2. Add a **Collapsible tabs** section.
3. Add your questions and answers as tab items.
