# Why aren't my collection filters showing up?

Adding variants or tags to your products doesn't make filters appear automatically. In Shopify OS 2.0 themes like Avante, filtering is managed through Shopify's native **Search & Discovery** app — you need to explicitly add each filter there.

***

**Quick fix:**

1. Install [Search & Discovery](https://apps.shopify.com/search-and-discovery) — it's free
2. Go to the app → **Filters** → Add filters (size, color, vendor, etc.)
3. Filters appear automatically on your collection pages

***

**Still not showing up?** Shopify only displays a filter if it would actually narrow down results. If every product in the collection shares the same value (e.g. all products come in the same sizes, all in stock), Shopify hides that filter — even with Search & Discovery installed and configured. This is expected behavior, not a bug.

For the full setup guide, see [Filters & search](../../collections-and-products/filters-and-search.md).
