# How do I create nested / dropdown menus?

Menu nesting is set in **Shopify Admin**, not in Theme Editor.

1. Go to **Shopify Admin → Online Store → Navigation**.
2. Open the menu you want to edit.
3. Drag items slightly to the right under a parent item.
4. Save the menu.

The theme automatically renders that structure as dropdowns or a mega menu, based on your header settings.
