# What's new in Avante

This changelog is written for merchants — not developers. Each entry highlights new features you can start using today, with links to the relevant documentation.

For the full technical release notes, visit the [Shopify Theme Store listing](https://themes.shopify.com/themes/avante/presets/avante).

---

## v14.0.0 — May 2026

### 🎨 Bloop preset redesigned
Bloop has been fully rebuilt as a starting point for food and drink stores. If you're in that space, install Bloop fresh from the Theme Store and start from the new layout.

### 🔡 Scalable fonts (rem units)
Fonts now scale based on the user's browser settings instead of fixed pixel sizes. No action needed — this improves accessibility automatically. Fine-tune sizes in [Theme settings → Typography](../theme-settings/typography.md).

### 🌈 Gradient headings and section backgrounds
Add gradient fills to headings and section backgrounds. Enable in [Theme settings → Colors](../theme-settings/colors.md).

### 📐 XLarge and XXLarge radius sizes
Two new corner radius options added. Set them in [Theme settings → Shapes](../theme-settings/shapes.md).

### ✨ Cascade animations in all drawers
Menus, cart drawer, store selector, and news drawer now animate with a cascade effect. Enable in [Theme settings → Animation](../theme-settings/animation.md).

### ⊞ Product grid gap controls
Control the spacing between product cards independently in product-related sections.

---

## v13.0.0 — March 2026

### 🏠 Fairmont preset
New preset designed for home and interior brands — editorial-style, polished out of the box.

### ↔️ Disable layout margins (edge-to-edge)
A new **Disable layout margins** setting lets images and sections extend to the full screen width. Find it in [Theme settings → Appearance](../theme-settings/appearance.md).

### 🔢 Section categories in Theme Editor
Sections in the Theme Editor are now grouped by category, making it easier to find what you need.

### 💫 Cascade animations
Cascade animations added to Menu drawer, Cart drawer, News drawer, and Store selector drawer. Enable per-drawer in [Theme settings → Animation](../theme-settings/animation.md).

### 🎛️ Compare-at price color
Customize the color of strikethrough prices. **Theme settings → Colors → Custom compare-at price color**.

### ⚓ Smooth anchor link scrolling
Clicking anchor links now scrolls smoothly. Enable in **Theme settings → Animation → Enable anchor links smooth scroll**.

### 🔍 Availability toggle in filters
A new filter option lets customers filter by "In stock" availability. Configure via the [Search & Discovery app](../collections-and-products/filters-and-search.md).

---

## v12.2.0 — January 2026

### 🖱️ Hover mega menu
A new mega menu type that opens on cursor hover instead of click — ideal for desktop-first stores with rich navigation. Add a **Hover mega menu** block in the [Header](../global-sections/header.md). See it live on the Vortex preset.

---

## v12.0.0 — October 2025

### 🎨 Adelle preset
New preset introduced. Install directly from the Shopify Theme Store to get the full demo layout.

### 📰 News Drawer section
A new full-screen drawer section for articles, announcements, or editorial content. Add it from the sections list in the Theme Editor.

### 🛍️ Shoppable Media Slider
A new section combining media (images/video) with clickable product tags. Add it from the sections list.

### 🎬 Stories Slideshow
A new vertical story-style slideshow section. Add it from the sections list.

### 🖼️ Horizontal Scrolling Banners
A new section for banner strips with horizontal scroll on mobile. Add it from the sections list.

### 🏷️ Discount code in Cart drawer
Customers can now enter discount codes directly in the cart drawer — no need to proceed to checkout first. Enable in [Theme settings → Cart drawer](../global-sections/cart-drawer.md).

---

## v11.0.0 — August 2025

### 🌄 Panorama preset
New preset added. Install from the Theme Store for the full demo.

### 🔤 Custom typography preset
A 9th typography preset giving you full independent control over every text style. Select **Custom** in [Theme settings → Typography](../theme-settings/typography.md).

### 🧭 Breadcrumbs
Breadcrumb navigation added for better user orientation and SEO. Enable in **Theme settings → Breadcrumbs**. Available on collection and product pages.

### 🎨 Color blending for product cards
Applies a subtle tint to very light product image backgrounds for a cleaner grid. Enable in [Theme settings → Product card → Color blending](../collections-and-products/collection-page.md).

### 🌍 Store Selector in menu drawer
The [Store Selector](../features/store-selector.md) can now be added as a block inside the menu drawer — letting customers switch stores without leaving navigation.

---

## v9.0.0 — May 2025

### 🌍 Store Selector
Let customers switch between regional stores from the header and product cards. Full setup guide: [Store Selector](../features/store-selector.md).

### ⚡ Link preloading on hover
Pages start preloading when a customer hovers over a link — making navigation feel nearly instant. Enable in **Theme settings → Animation → Preload links on hover**.

### 🌫️ Page fade-in effect
A smooth fade transition on page load. Enable in **Theme settings → Animation → Enable fade-in on page load**.

---

## v8.0.0 — March 2025

### 📱 Transparent mobile header
The transparent header effect now works on mobile too. Enable: **Theme settings → Appearance → Transparent header and sidebars → Enable on mobile**. Full guide: [Transparent header](../features/transparent-header.md).

### 🔢 Up to 2048 variants
Avante now supports up to 2,048 variants per product (Shopify's default is 100). No configuration needed — works automatically. More info: [Product variants](../collections-and-products/product-variants.md).

### 🖼️ Product card sliders on hover
Product cards can show a slider when hovered, letting customers browse images without opening the product page. Enable per section: **On hover → Show: Slider**.

### 🌐 Full RTL support
Right-to-left language support added across all layouts.

---

## v7.0.0 — January 2025

### 📅 Events section
A new section to showcase upcoming events with ticket CTAs and calendar add options.

### 🧩 Mega menu drawer
A full-screen off-canvas menu layout. Enable: **Theme settings → Navigation → Menu drawer style → Mega menu drawer**. More: [Header](../global-sections/header.md).

### ⏱️ Countdown timer on product page
Add urgency to time-sensitive promotions directly on the product page. Add a **Countdown timer** block in the Product overview section.

---

## v6.0.0 — November 2024

### 🚚 Free shipping progress bar
Show customers how close they are to free shipping. Enable in **Theme settings → Cart**. Works in both the cart drawer and cart page. More: [Cart drawer](../global-sections/cart-drawer.md).

### ⏱️ Countdown timers in banners
Countdown timer block added to Announcement Bar, Slideshow, Image Banner, and Media with text sections.

### ⭐ Highlight menu items
Mark specific navigation items (e.g. "Sale") with a highlight style. Configure in the Header, Main sidebar, or Mega menu block settings.
