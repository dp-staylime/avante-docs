# Search template

The **Search results** section is the core component of your store's `/search` page. It displays products, collections, pages, and blog posts that match the customer's search query. It shares many layout and merchandising settings with your Collection pages, ensuring a consistent browsing experience.

### General & Filtering

Control the main appearance of the search page and how customers filter the results.

* Heading size & color: Adjust the size of the main search heading and choose between a Solid color or a Gradient.
* Enable filtering and sorting: Toggles the filter sidebar/drawer and sorting dropdown. _(Note: The actual filter options are created and managed via the free Shopify Search & Discovery app)._
* Enable availability toggle: Adds a quick "In stock" switch to the top of the filters.
* Desktop filter style: Choose how filters are presented on desktop: Drawer, Horizontal, or Vertical.
* Drawer button style: Choose the design of the "Filter" button (Solid, Outlined, or Link).
* Description of filters with 'AND' operator: Explains the filter logic to customers (e.g., "Shows only results that meet all selected options").
* Tabs always open: If using the Drawer or Vertical style, you can type the exact names of specific filter tabs (separated by commas) to keep them expanded by default.

Grid LayoutConfigure how the search results are structured on the page.

* Items per page: Set how many results load initially (e.g., 48).
* Infinite scroll: Choose how to load more products. You can Disable it (to use standard 1, 2, 3 pagination), or set it to automatically load more results as the customer scrolls.
* Grid type (Desktop): Set the number of columns (e.g., Three columns).
* Mobile layout: Choose between One column or Two columns for mobile devices.
* Grid gap: Leave as Default or set a Custom spacing between the result cards.

Typography SizingFine-tune the text sizes specifically for the filter sidebar and tabs.

* Total products size: Size of the text showing the result count.
* Tab size: Size of the filter category names.
* Text size: Size of the filter options themselves. _(All options default to your global Typography preset, but can be overridden here)._

Product CardsThe search results grid inherits the full power of Avante's product cards. You can configure:

* Image ratio & Focal point: Ensure all product images align perfectly (e.g., Narrow portrait 2:3).
* On hover: Choose what happens when a customer hovers over a product: show a second image, enable a swipeable slider, or display Quick Add / Quick View buttons.
* Product card content: Show or hide the vendor, product rating, and adjust the price position.
* Show short description: Displays the `Product subtitle` metafield if filled out.
* Enable color pills & pickup availability: Show available color swatches and in-store pickup statuses directly in the search results.

Blog Post ResultsIf your search query matches articles from your blog, they will appear in the grid alongside products. You can customize their specific appearance:

* Show image: Toggle the blog post thumbnail.
* Image ratio & focal point: Adjust the crop of the article image (e.g., Portrait 4:5).
* Post metadata: Choose whether to show the publish Date, Author, and Number of comments.

Section Paddings

* Left and right (Desktop): You can use the store's default side padding (set in Theme settings → Appearance) or override it specifically for the search page.
* Top and Bottom: Adjust the vertical spacing above and below the search results for both Desktop and Mobile views.
